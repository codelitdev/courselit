## Mailing Capabilities

CourseLit offers two types of emails:

### 1. Broadcasts

Send one-off emails to selected recipients for quick updates or newsletters. Learn more about [broadcasts](/email-marketing/broadcasts).

### 2. Sequences

Automatically send a series of emails triggered by specific events in the system. Learn more about [sequences](/email-marketing/sequences).
